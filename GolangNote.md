## 数组(array)
#### 声明数组和声明切片的不同
```
// 创建有3 个元素的整型数组
array := [3]int{10, 20, 30}
// 创建长度和容量都是3 的整型切片
slice := []int{10, 20, 30}
```
***
## 切片(slice)
#### 如何计算长度和容量
```
对底层数组容量是k 的切片slice[i:j]来说
长度: j - i
容量: k - i
```
#### 修改切片内容可能导致的结果
```
// 创建一个整型切片
// 其长度和容量都是5 个元素
slice := []int{10, 20, 30, 40, 50}
// 创建一个新切片
// 其长度是2 个元素，容量是4 个元素
newSlice := slice[1:3]
// 修改newSlice 索引为1 的元素
// 同时也修改了原来的slice 的索引为2 的元素
newSlice[1] = 35
```
#### 使用3 个索引创建切片
```
// 将第三个元素切片，并限制容量
// 其长度为1 个元素，容量为2 个元素
slice := source[2:3:4]
```
#### 如何计算长度和容量
```
对于 slice[i:j:k] 或 [2:3:4]
长度: j – i 或3 - 2 = 1
容量: k – i 或4 - 2 = 2
```
